Quick Start Guide

1

Clone the Repository

Get the source code from GitHub.

git clone https://github.com/nairdahh/Anchorr.git
2

Configure Your Environment

Create a .env file from the example and fill in your API keys and server details.

cp .env.example .env
3

Run with Docker

Use Docker Compose to build and run the application in a container.

docker compose up -d --build
4

Set Up Webhooks

Add the webhook URL to your Jellyfin server for real-time notifications.

http://<your-host-ip>:<WEBHOOK_PORT>/jellyfin-webhook

For a fully detailed guide, including non-Docker setup and advanced configuration, please visit the GitHub page.
